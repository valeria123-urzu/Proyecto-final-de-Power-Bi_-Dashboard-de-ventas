# Proyecto-final-de-Power-Bi_-Dashboard-de-ventas
 Proyecto de Power BI enfocado en el análisis de ventas internacionales.
 
Este proyecto consiste en la elaboración de un dashboard interactivo en Power BI enfocado en el análisis de ventas globales. A través de diferentes visualizaciones y métricas, se busca facilitar la interpretación de la información comercial y apoyar en la toma de decisiones.

## Objetivo

Analizar el comportamiento de las ventas considerando distintos factores como productos, países, años y tiempos de entrega, con el fin de identificar tendencias, áreas de oportunidad y el desempeño general de las ventas.

## Indicadores principales
- Ventas totales  
- Clientes registrados
- Países analizados 

# Contenido del dashboard
El dashboard incluye diferentes apartados para visualizar la información de manera más clara:
 1. Ventas por producto
Se presenta un ranking con los 8 productos más vendidos, permitiendo identificar cuáles generan mayores ingresos y tienen mejor desempeño comercial.
2. Ventas por año
Se analiza la evolución de las ventas durante los años 1996, 1997 y 1998, observando los cambios y tendencias en distintos periodos mensuales.
3. Ventas por país
Se utiliza un mapa interactivo para mostrar la distribución geográfica de las ventas, destacando los países y regiones con mayor participación comercial, principalmente Europa y América.
4. Tiempo de entrega
Se comparan los tiempos de entrega por año mediante diferentes categorías de días (1, 4, 8, 16 y 23 días), permitiendo evaluar el desempeño logístico y su posible impacto en la satisfacción del cliente.

## Herramientas utilizadas
- Power BI  
- DAX (Data Analysis Expressions)  
- Modelado de datos  

## Uso del proyecto
Para visualizar el dashboard:

1. Descargar el archivo `.pbix`
2. Abrirlo en Power BI Desktop
3. Utilizar los filtros interactivos para explorar la información por:
   - País
   - Producto
   - Año
